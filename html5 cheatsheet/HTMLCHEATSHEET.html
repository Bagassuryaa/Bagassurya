Global Attributes

class Determine the CSS class to be used id Spesifies a unique id for an element
style Spsesifies inline CSS for elements title Spesifies additional information about the element

Special Attributes

src can be used in tags: <audio><embed><iframe><img> href can be used in tags: <a><link>
action can be used in tag: <form> autoplay can be used in tags: <audio><video>
alt can be used in tag: <img><input> colspan and rowspan can be used in tags: <td><th>

Special Attributes Function

src URL media href URL of the page to which you want to go
action determines where the form data is sent autoplay allows the audio or video to start playing automatically once the web page loads
alt alternative text if the element fails to load colspan and rowspan number of table cell and row columns to be filled
